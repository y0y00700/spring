관심사의 분리
ex) 
배우는 본인 역할인 배역 수행에만 집중.
배역이 바뀐 다른 주인공으로 교체되더라도 배우는 해당 배역 수행만 담당.
담당 배우를 섭외하고 지정하는 책임을 담당하는 기획자가 별도로 존재 해야 함.
공연 기획자를 만들고 배우와 공연 기획자의 책임을 분리한다.
